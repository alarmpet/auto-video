# Visual Context Mismatch Root Cause Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make generated images match each narration chunk's concrete sentence context, biblical event, core object/action keywords, and emotional beat instead of producing generic dark biblical figures, lamps, tents, and roads.

**Architecture:** Add a `visual-beat` layer before context-card generation. The beat layer preserves the exact narration chunk, extracts concrete anchors from biblical citations and modern psychology sentences, classifies whether the scene is a story event, scripture quote, inner psychology, or metaphor, then forces prompt generation and QA gates to use those source-derived anchors. The current self-referential prompt checks are replaced with source-to-prompt checks and a lightweight image QA report using generated keyframe contact sheets.

**Tech Stack:** Node.js ES modules, existing `scripts/build_segmented_storyboards.mjs`, existing `scripts/lib/sentence-grounded-visual-timeline.mjs`, existing `scripts/lib/scene-context-card.mjs`, Hermes manual storyboard format, FFmpeg for contact sheets.

## Investigation Summary

The user complaint is valid for the latest generated video:

- Export: `C:\Users\petbl\auto-video\exports\gguljam-bible-inferiority-defense-saul-david-10min-001`
- Hermes job: `C:\Users\petbl\hermes-studio\hermes-local\outputs\job-2026-07-03T10-13-48-649Z`
- QA contact sheet: `C:\Users\petbl\auto-video\exports\gguljam-bible-inferiority-defense-saul-david-10min-001\final\qa-frames\keyframe_contact_sheet.jpg`

Observed image pattern:

- Most generated images are a lone man, lamp, tent, dark road, doorway, table, or fire.
- Concrete Saul-David story anchors are mostly absent: Saul, David as contrast, women singing, thousands/ten-thousands comparison, spear, harp, palace, robe edge, cave encounter.
- Modern psychology anchors are also too abstract: comparison, insecurity, defense, shame, recognition anxiety become generic posture rather than visible situations.

Verified code causes:

- `scripts/build_segmented_storyboards.mjs` creates `groundedScenes` with narration, but writes `visual-timeline.json` without `narration`. Downstream QA loses source text.
- `scripts/lib/scene-context-card.mjs` has `David` in `CHARACTER_RULES` but not `Saul`.
- `scripts/lib/scene-context-card.mjs` lacks detail lexicon entries for `창`, `수금`, `왕궁`, `여인들`, `만만`, `천천`, `겉옷`, `굴`, `사울`.
- `inferBiblicalEvent()` has Elijah/Jacob/Cain/Adam rules but no Saul-David rules.
- `check_visual_grounding_timeline.mjs` checks whether generated prompt contains terms generated by the same context card. This validates its own output, not whether the prompt truly matches the source sentence.
- Hermes event log contains `visual_lineage_hangul_warning`: source narration had Hangul but downstream promptbook lost it. That is a traceable warning, but no auto-video gate currently fails on it.
- `keyframe_critic_started` appears without completion in the Hermes job event log, so there was no final image-context critic report for the rendered 27 keyframes.

Root cause:

The pipeline is sentence-timed but not sentence-grounded enough. It chunks the narration correctly, then compresses rich Korean source text into generic psychology rules and a small detail lexicon. The QA gates prove the prompt contains those generic generated terms, so the system passes even when the actual sentence's interesting content is missing.

## Review Incorporation Decisions

Review source: `docs/superpowers/plans/2026-07-06-visual-context-mismatch-root-cause-plan-review-report.md`

Accepted refinements:

- Narrow the Saul-David victory song rule so it only matches song/comparison language or `1 Samuel 18:7`, not any broad `Saul ... David` co-occurrence.
- Keep a broad Saul-David fallback rule only at the bottom of `SAUL_DAVID_RULES`, because `Array.find()` returns the first match.
- Require either `Saul` or `David` in the suspicion/watch/fear rule so generic psychology narration does not become a Saul-David scene.
- Remove `lyre` from the spear-threat rule. Lyre belongs in anchor vocabulary, but it must not turn peaceful music scenes into spear-threat visuals.
- Make `check_keyframe_context_sheet.mjs` calculate its tile layout from timeline length instead of using an old fixed 3-by-9 sheet.
- Use `scene.requiredPromptTerms` from `visual-timeline.json` as the grounding gate's primary source instead of requiring a second `visual-beats.json` load.

Rejected items: none. The review findings are technically consistent with the current planned implementation and would prevent real false positives or brittle verification.


```js
  [/수금/u, "lyre"],

  [/두려/u, "fear of losing place"],
  [/인정/u, "need for recognition"],
  [/비교/u, "comparison anxiety"],

const FORBIDDEN_GENERIC_ONLY = [
  "oil lamp",
  "family tent",
  "empty sleeping mat",
  "generic lone man",
  "generic dark road",
];

export function classifyVisualBeat(narration = "") {
  const text = String(narration || "");
  const eventRule = SAUL_DAVID_RULES.find((rule) => rule.re.test(text));
  if (eventRule) return eventRule.kind;
  const psychologyRule = PSYCHOLOGY_RULES.find((rule) => rule.re.test(text));
  if (psychologyRule) return psychologyRule.kind;
  return "reflective_bridge";
}

export function extractVisualAnchors(narration = "") {
  const text = String(narration || "");
  const anchors = [];
  for (const [re, value] of ANCHOR_LEXICON) {
    if (re.test(text) && !anchors.includes(value)) anchors.push(value);
  }
  return anchors;
}

export function buildVisualBeat({ narration = "", order = 1 } = {}) {
  const text = String(narration || "");
  const eventRule = SAUL_DAVID_RULES.find((rule) => rule.re.test(text));
  const psychologyRule = PSYCHOLOGY_RULES.find((rule) => rule.re.test(text));
  const rule = eventRule || psychologyRule || {
    kind: "reflective_bridge",
    event: "quiet reflective transition grounded in the previous biblical idea",
    characters: [],
    required: ["visible emotional distance", "quiet relational tension"],
  };
  const anchors = extractVisualAnchors(text);
  const requiredPromptTerms = unique([
    ...rule.required,
    ...anchors.filter((anchor) => !rule.required.some((term) => term.includes(anchor))),
  ]).slice(0, 8);
  return {
    order: Number(order) || 1,
    narration: text,
    kind: rule.kind,
    event: rule.event,
    characters: rule.characters,
    anchors,
    requiredPromptTerms,
    forbiddenGenericOnlyTerms: FORBIDDEN_GENERIC_ONLY,
  };
}

function unique(values) {
  return [...new Set(values.filter(Boolean))];
}
```

Create `scripts/check_keyframe_context_sheet.mjs`:

```js
#!/usr/bin/env node
import { existsSync, mkdirSync, readFileSync, writeFileSync } from "node:fs";
import { join } from "node:path";
import { execFileSync } from "node:child_process";

const args = parseArgs(process.argv.slice(2));
if (!args.segmentDir || !args.keyframesDir || !args.outDir) {
  console.error("Usage: node scripts/check_keyframe_context_sheet.mjs --segment-dir <dir> --keyframes-dir <dir> --out-dir <dir>");
  process.exit(2);
}

mkdirSync(args.outDir, { recursive: true });
const timeline = JSON.parse(readFileSync(join(args.segmentDir, "visual-timeline.json"), "utf8")).scenes || [];
const prompts = parseStoryboard(readFileSync(join(args.segmentDir, "hermes-manual-storyboard.md"), "utf8"));
const failures = [];
const scenes = timeline.map((scene, index) => {
  const keyframePath = join(args.keyframesDir, `scene_${String(index + 1).padStart(2, "0")}.png`);
  if (!existsSync(keyframePath)) failures.push(`missing_keyframe:${keyframePath}`);
  return {
    order: scene.order || index + 1,
    narration: scene.narration || prompts[index]?.narration || "",
    requiredPromptTerms: scene.requiredPromptTerms || [],
    prompt: prompts[index]?.prompt || "",
    keyframePath,
  };
});

if (timeline.length !== prompts.length) failures.push(`timeline_prompt_count_mismatch:${timeline.length}/${prompts.length}`);

const sheetPath = join(args.outDir, "keyframe-context-sheet.jpg");
if (!failures.length && timeline.length > 0) {
  const cols = Math.min(5, Math.max(1, timeline.length));
  const rows = Math.ceil(timeline.length / cols);
  execFileSync("ffmpeg", [
    "-y",
    "-framerate", "1",
    "-start_number", "1",
    "-i", join(args.keyframesDir, "scene_%02d.png"),
    "-frames:v", String(timeline.length),
    "-vf", `scale=320:180,tile=${cols}x${rows}`,
    sheetPath,
  ], { stdio: "ignore" });
}

const report = {
  ok: failures.length === 0,
  segmentDir: args.segmentDir,
  keyframesDir: args.keyframesDir,
  sheetPath: existsSync(sheetPath) ? sheetPath : null,
  failures,
  scenes,
};
writeFileSync(join(args.outDir, "keyframe-context-report.json"), JSON.stringify(report, null, 2), "utf8");
console.log(JSON.stringify({ ok: report.ok, failures, sheetPath: report.sheetPath }, null, 2));
process.exit(report.ok ? 0 : 1);

function parseStoryboard(markdown) {
  const lines = String(markdown || "").split(/\r?\n/);
  const blocks = [];
  for (let index = 0; index < lines.length; index += 1) {
    const label = lines[index].trim();
    if (!/^\[.*\]$/.test(label)) continue;
    const narration = label.slice(1, -1);
    const promptLine = String(lines[index + 1] || "").trim();
    blocks.push({ narration, prompt: promptLine.split(/\s+\/\s+/)[0] || promptLine });
  }
  return blocks;
}

function parseArgs(argv) {
  const parsed = {};
  for (let i = 0; i < argv.length; i += 1) {
    if (argv[i] === "--segment-dir") parsed.segmentDir = argv[++i];
    else if (argv[i] === "--keyframes-dir") parsed.keyframesDir = argv[++i];
    else if (argv[i] === "--out-dir") parsed.outDir = argv[++i];
  }
  return parsed;
}
```
