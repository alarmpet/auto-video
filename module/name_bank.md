# name_bank.md

> 인물 작명 은행. 4-B 캐릭터 프로필 작성 시 사용.
> **자유 생성 금지. 여기서만 조회한다.** 모델 즉흥 작명이 반복(연이·도화·분이·무휼·막동…)의 원인.
> **쓴 이름은 메모리로 최근 N편과 겹치지 않게 한다.**
> **핵심 규칙 1 — 발음 (★ 최우선. TTS·청취용 ★):** 소리 내 읽을 이름이다. **부르기 쉬운 것 우선.** 자음이 빡빡하거나 받침이 겹쳐 혀가 꼬이는 이름(처겸·휘원·형렬류)은 후순위. **성씨+본명을 합쳐 발음했을 때 굴러가야 한다** (✗변경석·✗명정월·✗방인후처럼 빡빡하게 쌓지 말 것). 특히 주인공·주요 인물은 가장 쉬운 이름으로.
> **핵심 규칙 2 — 반복 회피:** 받침 없는 부드러운 2음절(연이·도화류)로 몰리지 말 것. 끝음절·받침을 분산한다. 단, **규칙 1을 어기면서까지 특이하게 가지 말 것** — 또렷하되 쉬운 게 정답.
> 신분에 맞는 이름을 쓰면 반복도 줄고 고증도 자연스러워진다 (천민=막이름 / 사대부=한자 항렬형 / 여성 양반=택호).
> **부르는 법:** 전체 본명(성+이름)은 첫 소개 때 한 번만. 평소엔 이름(정월)·택호(해주댁)·직함(변 부자·봉 객주)으로 부른다 — 이게 듣기 편하다.

## 4. 양반·사대부

**작명 규칙(남성):** 한자 2음절 항렬형. 의미자(서·헌·원·진·현·후·겸 등) 조합. 서사에서는 본명 또는 호(號)로 부른다.

### 남성

### 여성
> ★ 사대부가 여성은 본명을 거의 부르지 않는다. **기본은 택호(친정 지명+댁)나 호칭.** 본명은 서사상 꼭 필요할 때만 쓴다.

- 호칭·택호 (우선): 아씨, 소저, 부인, 마님 / 안동댁, 청송댁, 해주댁, 나주댁, 동래댁, 광주댁, 전주댁, 진주댁
- 본명 (필요시): 정인, 명숙, 서온, 인선, 정온, 숙온, 명온, 효온, 단정, 정렬, 윤정, 명화, 정아, 숙영, 윤희

## 7. 왕실·궁중

> ★ 임금·왕비·세자는 **이름(휘)을 직접 부르지 않는다.** "임금·주상·전하·상감 / 중전·왕비·마마 / 세자·동궁"으로 칭한다. 본명 노출은 피한다.

- 상궁·나인 (필요시): 자선, 옥교, 난정, 보연, 명주, 한설, 윤복
- 내관 (필요시): 처선, 득남, 상번

## 8. 성씨 풀 (김·이·박 편중 차단 + 발음 쉬운 것 우선)

> 한 채널 안에서 같은 성씨 돌려쓰지 말 것. 단 **주인공·주요 인물은 읽기 쉬운 성씨**를 우선한다. 성씨+본명을 합쳐 발음해 보고 빡빡하면 더 쉬운 성씨로 바꾼다.

- **읽기 쉬운 성씨 (주요 인물 우선):** 김, 이, 박(가끔), 한, 윤, 서, 황, 명(明), 방(方), 도(都), 구(具), 추(秋), 표(表), 봉(奉)
- **드물고 어려운 성씨 (조연·단역만 가끔, 주인공엔 지양):** 견(甄), 빈(賓), 어(魚), 변(邊), 호(扈), 음(陰), 좌(左), 범(范), 갈(葛), 옥(玉)
- **복성 (드묾, 특별할 때만):** 선우, 남궁, 제갈, 황보, 사공, 독고, 동방, 서문

## 사용 규칙 (요약)
1. 인물의 신분·성별 확인 → 해당 칸에서만 이름 조회.
2. 사대부 여성은 본명 대신 택호·호칭 우선. 왕실은 직책으로 칭함.
3. 성씨는 8번 풀에서 분산 선택하되 **주요 인물은 읽기 쉬운 성씨 우선** (김·이·박 연속 금지, 성+이름 합쳐 발음 확인).
4. 고른 이름은 메모리로 최근 N편과 겹치지 않게 한다.
5. 은행이 동나면 각 칸의 **작명 규칙**으로 확장하되, 0번 차단 패턴(부드러운 2음절 쏠림)으로 회귀 금지.
