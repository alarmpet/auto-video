# 참고 2(slim): 캐릭터 말투·문체·신분 증표

## 캐릭터별 말투 매핑

| 유형 | 어미 | 특징 |
|---|---|---|
| 주인공(선량) | ~하옵니다, ~이옵니다 | 또박또박, 담담, 짧게 |
| 악역 | ~하시오, ~란 말이오 | 고압적, 비꼼, 협박 |
| 권력자 | ~하라, ~이니라 | 위엄, 간결 |
| 조력자/노인 | ~느니라, ~이니라 | 깊이, 은유 |
| 구경꾼 | ~대, ~래 | 수군거림, 감탄 |
| 추리 주인공 | ~이오, ~하시오 | 논리적, 질문형 |
| 가족 갈등 | 반말/존대 오감 | 감정 폭발, 진심 못 함 |

## 비하 호칭 제한 (★최대 2회★)

"~년", "이 년" 등 대본 전체 최대 2회. 위압감은 호칭 반복이 아니라 **구체적 위협·행동**으로.
대체: "감히 천한 것이", "건방진 것", "네 따위가", "당장 꺼지지 못할까", "가만두지 않겠다".

## 나레이션 문체 (★핵심★)

기본: **`~습니다` ↔ `~지요` 교차** (둘이 80% 차지해도 OK). 지루하지 않은 비결은 어미 다양성이 아니라 **리듬 장치 3가지.**

- **장치 1 — 대사 삽입:** 나레이션 500자 이상 연속 금지. 대사가 리듬을 깸.
- **장치 2 — 수사 질문(10문장당 1회):** "그런데 이게 웬일입니까?" / "학수의 마음이 어땠겠습니까." / "누가 이런 일이 벌어질 줄 알았겠습니까."
- **장치 3 — 짧은 전환 문장:** "그때였습니다." / "야반도주였습니다." / "설상가상이었습니다." / "그런데 이상한 일이 벌어졌습니다."

**어미 룰:** ~습니다 계열 3문장 연속 금지(2연속 OK). 모든 어미(~지요·~어요·~을까요·~군요·~네요…) 인접 동일 2회 초과 금지(3연속 금지). 나레이션만, 대사 제외.

## 기본 문체
- 문장 15~25자 권장(TTS). 감탄사: 세상에·아이고·허허·참으로. 시간 전환: "그때였습니다"·"세월이 흘러".

## 분위기 묘사 (대표)
- 긴장: "칠흑같이 어두운 밤이었습니다." / "싸늘한 바람이 볼을 스쳤지요."
- 슬픔: "가슴이 찢어지는 듯했지요."
- 추리: "무언가 석연찮았습니다." / "그 말에는 빈틈이 있었지요."

## 외모 묘사 (대표)
- 초라함: "누더기를 걸쳤지만 눈빛만은 죽어 있지 않았습니다."
- 변신: "때를 씻기니, 세상에, 눈처럼 하얀 살결이 드러났습니다."

## 신분 증표 풀 (★옥패 편중 방지★)

**같은 유형 연속 3편 이상 금지. 복합 증명(2가지+ 조합) 권장.**

| 유형 | 예시 |
|---|---|
| 신체 표식 | 등의 점 세 개, 발바닥 붉은 반점, 어깨 화상, 손목 쌍점, 이마 흉터 |
| 의복/천 | 태보 반쪽, 수놓은 천 조각, 문양 노리개, 어머니가 짠 삼베 |
| 문서/서찰 | 밀봉 유서, 호적 기록, 사찰 기탁 문서, 관아 출생 기록 |
| 가문 기물 | 가문 비녀/반지/도장, 깨진 거울 반쪽, 어머니 은장도 |
| 기억/증언 | 유모 증언, 특정 자장가, 유모만 아는 버릇 |
| 기술/습관 | 가문 검법, 특정 요리법, 말버릇, 가문 글씨체 |
