# 참고 3(slim): 인트로·제목 가이드

## 제목 6가지 패턴

| 패턴 | 예시 |
|---|---|
| 충격 행동 + 반전 궁금증 | 장터에서 아기를 100냥에 사온 과부, 그 아이의 정체는? |
| A vs B 대비 | 큰 며느리는 땅 갖고 막내 며느리는 시어머니를 가졌다 |
| 상황 + 미완성 반응 | 세자빈 간택에 거지 차림으로 나온 처자, 모두 비웃었는데.. |
| 은혜 행동 + 그날 밤 결과 | 흰 뱀을 구한 농부, 그날 밤 문 앞에 나타난 소녀 |
| 의외의 상속·선택 (역설) | 황금 대신 돌멩이를 고른 바보 총각 |
| 신분역전 + 운명 | 거지 소년을 거둔 과부, 10년 후 벌어진 일 |

제목 끝에 ` | 야담 옛날이야기 민담 전설 설화` 부착.

## 인트로 자가검증 5문항 (★의무★)

인트로 작성 후 아래 5문항. **하나라도 "예"면 폐기 후 재작성.**

1. 결말을 짐작할 수 있는가?
2. 주인공의 숨겨진 정체·신분을 알 수 있는가?
3. 범인·진범이 누구인지 감이 오는가?
4. 갈등이 해결된다는 것을 알 수 있는가?
5. 제목 속 행동의 동기·결과·정체·반전이 노출됐는가? (행동 자체를 1문장으로 보여주는 것은 허용. 그 행동의 '왜·어떻게·누구·결말'만 노출 금지.)

## 인트로 대사 선택 공식

대사 출처: ③ 설정 ~ ⑥ 2막 진입 구간. 공식: **[극단적 행동·발언] + [그 이유는 아직 모름]**

유형: 위기 선언형(악역이 주인공 위협) / 금지·경고형 / 부조리·충격형(상식 밖 선언) / 미스터리 제시형 / 갈등 폭발형(가까운 사이 분노).

## 인트로 6문장 포맷 (200~350자)

| 문장 | 룰 |
|---|---|
| 1 (파격 대사) | 큰따옴표, 날것의 감정(원망·분노·절규·폭로). 공손체 금지. 구체적 숫자·물건·행동 |
| 2~4 (압축 상황) | 한 문장에 정보 여럿. 결과·결말 절대 금지. "지금 이 순간"만 |
| 5 (궁금증) | "그런데"로 시작. 닫힌 질문 금지(결말 노출 금지) |
| 6 (구독+시작) | 고정: "구독과 좋아요는 더 좋은 이야기를 만드는 힘이 됩니다. 그럼 지금부터..." |

닫힌 질문 금지:
❌ "과연 복이는 진짜 대감의 딸이었을까요?" (딸이라는 걸 노출)
✅ "그런데 이 아이에게는 아무도 모르는 사연이 하나 숨겨져 있었습니다."

## Good 예시 (2종 — 톤 참고)

**권선징악**
"쌀 석 섬이면 된다. 석 섬만 갖고 오면 이 계집애를 데려가도 좋다."
장터 한복판에서 열두 살 먹은 아이가 쌀자루 값에 팔리고 있었습니다.
아이의 팔뚝에는 때에 절은 삼베 조각이 묶여 있었고, 눈빛만은 죽어 있지 않았습니다.
그런데 이 아이에게는 아무도 모르는 사연이 하나 숨겨져 있었습니다.

**미스터리·추리**
"금궤 열 상자가 사라졌다. 문은 안에서 잠겨 있었고, 열쇠는 나만 갖고 있었다."
원님의 얼굴이 하얗게 질려 있었습니다.
관아 금고의 철문에는 흠집 하나 없었고, 창문은 사람이 빠져나갈 수 없는 크기였습니다.
그런데 이 사건에는 금궤보다 더 위험한 비밀이 감춰져 있었습니다.
