## 비트별 룰

| 비트 | 핵심 룰 |

| ① 오프닝 이미지 | "옛날 옛적 [구체 지역]에…" 시작. 주인공 외적 상태 + 결핍. ⑮와 대척점 |
| ② 주제 명시 | 누군가 주인공에게 주제 대사. 주인공 무시·이해 못함. 엔딩에서 울림 |
| ③ 설정 | 일상 세계 + 주요 인물 소개 + 약점/결핍 에피소드 + 악역 암시 |

| ⑪ 절망의 순간 | 모든 것을 잃음. 무언가 죽어야 함(사람/관계/신념/상징물) |

## ★ 2시간(strict) 핵심만 ★

2시간이 짧아지는 주범은 ⑧⑩⑭의 요약 본능. 펼쳐 쓸 것.
- **⑧ 재미와 놀이 (8,280자):** 에피소드 6개+, 각 1,200~1,500자. 장소·상대·상황 모두 다르게(같은 장소 2회 이하). 각 에피소드 = 도입→전개(시련·속임수)→마무리.
- **⑩ 악당의 역습 (5,520자):** 3단계를 독립 장면으로. 약점 파악 1,500자+ / 이간질(주변 인물 실제로 흔들림, 대사 중심) 1,800자+ / 최후 수단 1,800자+. 단계 사이 장소·시간 전환.
- **⑭ 피날레 (5,980자):** 5단계 각 1,000자+. 대면→진실증명(증표·논리, 관중 반응)→악역몰락(가장 긴 단계, 폭로→변명실패→측근배신→심판)→형벌(구경꾼 반응)→보상(주변 반응+감동).
- **⑦ (2,760자):** 독립 에피소드 2개(조력자 첫 유대 / 가르침).
- **③ (5,060자):** 주인공 일상 2장면 + 악역 암시 1 + 주변 인물 소개 1.

## 고정 마무리 멘트 (정확 일치)

```
다음 영상을 빠르게 만나보시려면 좋아요와 구독을 눌러주세요.
지금 화면에 나오는 더 재미있는 영상들도 함께 해주세요.
그럼 모두 행복한 하루 보내세요. 감사합니다.
```

## 챕터 분할

**1.5시간 (35,000자, 7챕터, 디폴트):** 1[인트로~②] / 2[③] / 3[④~⑥] / 4[⑦~⑧전반] / 5[⑧후반~⑨] / 6[⑩~⑪] / 7[⑫~엔딩], 각 ~5,000.

**1시간 (25,000자, 5챕터):** 1[인트로~②] / 2[③~④] / 3[⑤~⑧전반] / 4[⑧후반~⑩] / 5[⑪~엔딩].

**2시간 (46,000자, 9챕터, strict):** 1[인트로~②] / 2[③전반] / 3[③후반~④] / 4[⑤~⑥] / 5[⑦~⑧전반] / 6[⑧후반] / 7[⑨~⑩전반] / 8[⑩후반~⑬] / 9[⑭~엔딩]. **챕터 바닥 4,410자.**
- ⑧ 총량: 챕터5후반+6 = 8,280 ±15%(7,038~9,522). ⑩ 총량: 챕터7후반+8전반 = 5,520 ±15%(4,692~6,348).

## 장르 체크리스트 (공통: ①↔⑮ 대척점, 피날레 5단계)

- **권선징악:** 주인공 굴욕 2회+ / 신분·진실 증명 장치 / 능력·지혜 증명 / 악역 음모 3단계+
- **귀신·도깨비:** 으스스 분위기 / 초자연 첫 등장 긴장 / 주인공 선택·시험 / "알고 보니~" 반전 / 욕심 vs 정직
- **해학·풍자:** 속임수 단계 상세 / 호구 속는 과정 / 구경꾼 반응 / 통쾌한 반전
- **사랑·비극:** 운명적 만남 / 사랑 맹세 / 애틋한 이별 / 약속 물건 3회+ / 기다림
- **역사 인물:** 실존 특징 / 역사 배경 / 지혜·용기 에피소드 / "오늘날까지 전해져…"
- **미스터리:** 사건 / 단서 3개+(1개 미스디렉션) / 용의자 3명+(각 동기) / 추리(관찰→가설→검증) / 의외 반전
- **가족·성장:** 핵심 갈등 2회+ / 진심 못 함 / 숨겨진 희생 / 성장 계기 / 화해·재회 / 가족 상징물 3회+
