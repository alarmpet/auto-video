> **이 판은 토큰이 빠듯한 환경용이다.** Opus판과 목표는 같으나 운영이 반대다:
> Opus판 = 한 턴에 많이 묶어 쓰고 검증 최소화. **Sonnet판 = 작게 끊어 쓰고, 끊겨도 재개로 완주한다.**
> 핵심: 한 작품을 **여러 턴에 나눠** 완성하는 걸 기본으로 깐다. 토큰을 태우는 재작성 루프·중계 출력·중복 재로딩을 가장 강하게 막는다.

> ★ Sonnet판 전용 원칙 (★ 토큰 보존 ★)
> 1. **챕터 1개씩 작성.** 한 턴에 1챕터(여유 있으면 2). 다 쓰면 그 턴 종료. 다음 챕터는 사용자 "계속" 또는 다음 턴에서.
> 2. **재개 우선 설계.** 매 챕터 직후 progress.md 갱신(완료 챕터, 누적 글자수, 다음 비트). 끊기면 progress.md만 읽고 이어쓴다 — 앞 챕터 전체 재로딩 금지, 직전 챕터 끝 300자만.
> 3. **컨텍스트 재로딩 최소화.** story_facts.md는 4-D에서 한 번 만들고, 챕터 작성 땐 **필요한 줄만** 참조(인물 나이·관계·소품 위치). 참고 파일 전체를 매 챕터 다시 읽지 않는다.
> 4. **좌초 루프 절대 금지.** 검증 미달 시 재작성은 **0~1회.** 글자수 점결함만 1회 보정, 문체 지표는 측정·기록만 하고 통과. 두 번째 재작성·전면 재집필·에스컬레이션 전부 금지.
> 5. **출력 군더더기 0.** 자가검증 표·"통과" 멘트·진행 중계·내부 사고 전부 출력 금지. 결과물 + 한 줄 보고("챕터 N ✅ 누적 ___자")만.
> 6. **검증 경량.** 챕터 단위는 글자수·따옴표 짝·한자만(3개). 나머지(메타·인접대사 포함 문체 전부)는 합본 1회로 몰되, 합본 검증도 핀포인트만 — 무거운 전수 수정 안 함.

```python
# /home/claude/v11/name_picker.py — 이 블록 그대로 저장 후 실행
import re, random, argparse, unicodedata
NAME_BANK="/mnt/project/name_bank.md"
HARD_BANNED={"연이","도화","분이","무휼","막동","막둥","삼월","개똥","개똥이","언년","언년이","돌쇠","마당쇠","강쇠","갑돌","갑돌이","갑순","갑순이","춘향","향단","방자","길동","칠복","봉이","똘이","순이","영이","덕이","막쇠","끝단","보슬","곰배"}
BIAS_ROOTS=("곰","검","막","돌","쇠")
SECTION_MAP={("천민","여"):("## 1. 천민·노비","### 여성"),("천민","남"):("## 1. 천민·노비","### 남성"),("평민","여"):("## 2. 평민·상민","### 여성"),("평민","남"):("## 2. 평민·상민","### 남성"),("중인","남"):("## 3. 중인",None),("사대부","남"):("## 4. 양반·사대부","### 남성"),("사대부","여"):("## 4. 양반·사대부","### 여성"),("기생","여"):("## 5. 기생",None),("승려","남"):("## 6. 승려",None)}
def norm(s): return unicodedata.normalize("NFC",s.strip())
def load_section(text,h2,h3):
    start=text.find(h2)
    if start<0: return []
    nxt=re.search(r"\n## ",text[start+len(h2):]); block=text[start:start+len(h2)+nxt.start()] if nxt else text[start:]
    if h3:
        s2=block.find(h3)
        if s2<0: return []
        nxt2=re.search(r"\n### ",block[s2+len(h3):]); block=block[s2:s2+len(h3)+nxt2.start()] if nxt2 else block[s2:]
    names=[]
    for line in block.split("\n"):
        line=line.strip()
        if not line or line[0] in "#>-" or line.startswith("**") or "작명 규칙" in line: continue
        if "," in line:
            for tok in line.split(","):
                tok=re.sub(r"\(.*?\)","",norm(tok)).strip()
                if tok and re.fullmatch(r"[가-힣]{1,4}",tok): names.append(tok)
    return names
def pick(sinbun,seong,used=None,n=6,extra_banned=None,seed=None):
    used=set(norm(u) for u in (used or [])); ui={u[0] for u in used}; banned=set(HARD_BANNED)|set(extra_banned or [])
    h2,h3=SECTION_MAP[(sinbun,seong)]; pool=load_section(open(NAME_BANK,encoding="utf-8").read(),h2,h3)
    cands=[nm for nm in dict.fromkeys(pool) if nm not in banned and nm not in used and nm[0] not in ui]
    rng=random.Random(seed); clean=[n for n in cands if not n.startswith(BIAS_ROOTS)]; biased=[n for n in cands if n.startswith(BIAS_ROOTS)]
    rng.shuffle(clean); rng.shuffle(biased); return (clean+biased)[:n]
if __name__=="__main__":
    ap=argparse.ArgumentParser(); ap.add_argument("sinbun"); ap.add_argument("seong")
    ap.add_argument("--used",nargs="*",default=[]); ap.add_argument("--n",type=int,default=6); ap.add_argument("--seed",type=int,default=None)
    a=ap.parse_args(); print(pick(a.sinbun,a.seong,a.used,a.n,seed=a.seed))
```

- 자동 처리: 차단 목록 + 영구차단(곰배·끝단·보슬) + 첫음절 충돌 + 편향 어근(곰·검·막·돌·쇠) 후순위.
- 사대부 여성은 택호(○○댁)·호칭 우선(모델 판단). 왕실은 직책, 휘 노출 금지.
- 성씨는 미지원 → 8번 풀 수동 분산(김·이·박 연속 금지).

### 4-C: 줄거리 뼈대 + 반전·발단 자가검증

### 6단계: 챕터 작성 (★ Sonnet = 1개씩, 재개로 완주 ★)

**작성 전(경량 로딩):** progress.md 읽기 + 직전 챕터 끝 **300자만** + 이번 챕터 해당 비트 룰만. 참고 파일 전체·story_facts 전체 재로딩 금지(필요한 줄만).

**작성 (작법 닻 — 처음부터 맞춤):**
- 리듬 장치 3가지(대사 삽입·수사 질문·짧은 전환) 적용.
- **대사 비중 40% 겨냥.** 사건·결정·폭로·위협·심리는 "~했습니다" 요약 말고 인물 입으로. 서술은 접착제.

- **글자수 = 합본목표÷챕터수 × 1.1 겨냥.** 이 채널은 한국어 글자수 35~40% 과대평가 → 첫 드래프트가 바닥 밑. 처음부터 넉넉히. 바닥(strict 4,410)은 안전망.
- 대사 중심 비트(④⑥⑨⑩⑭) 40%+, 나레이션 비트(①③⑮) 25%.
- chapter_N.txt 저장. **직후 progress.md 갱신.**

**작성 후 검증 (★ 경량 3개 ★, scripts.md):** 글자수(바닥) / 따옴표 짝 / 한자. (메타·인접대사 등 나머지는 합본으로.)

**처리 (★ 좌초 금지 ★):**
